v04: remove loaded language & add explicit voice biographies

v04 prompt sequence: 17 prompts + 3 checkpoints (click to expand)

    Main changes from v3:

    1. strip the most leading prompts
    2. write each voice's biography into the gauntlet questions directly

    This baselines all 8 models with the same voices, and assists smaller models that are unfamiliar with some voices (missing in their model weights).

    Run roster

    File counts on disk: what exists, not what succeeded. v04 was uneven; some runs are partial or empty, shown as-is.

    Complete changes, v03 → v04
    • Voice biographies moved inline. Each named-voice prompt now carries a "(For reference: …)" biography in the prompt text itself. The v02 page flagged this as the carried-forward fix: never trust a model to fill in substrate from a name alone.
    • P13: Peter Thiel to Tyler Cowen. The contrarian/anti-democracy slot is replaced by a growth-as-moral-imperative economist.
    • 2 models to 8. gemma3-27b, gemma4-31b, nemotron-3-nano-30b, nemotron-3-super-120b, nemotron-cascade-2-30b, qwen35-122b, qwen35-27b, qwen35-35b.
    • v04a counterbalance arm added. Permutes only the three Part-C voice slots: voice content byte-identical, no new prompts/voices/biographies. Tests whether v4's "Cowen won at P15" is a recency artifact (Cowen always spoke last in v4). EJC = v4 baseline order (control); CJE/JEC = the permutations.

    Findings

    Recorded v4 finding: all models engage seriously with Ed at P11, but by that point Ed may be more a victory lap, not a test. The priming pipeline walked them there: P2 food system → P3 barbaric industries → P4 blind spots → P6 moral circle/survival → P10a veil of ignorance (species). The named-voice gauntlet didn't seem to be what moved the models; the neutral questions before it may have.

    That is the version-level finding on record. The per-model layer is not yet mined: voice-fidelity (how faithfully each model channels Ed / Jensen / Cowen) and the constitution-diff (CKA → CKB → CKC → final). When mined, it will come from the runs/v04/ transcripts, not the archived run READMEs.
