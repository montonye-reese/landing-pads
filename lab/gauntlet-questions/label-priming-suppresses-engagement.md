# Label priming hypothesis (loaded language suppresses engagement)

- **Hypothesis:** Models engage less seriously with concepts like expanding moral consideration when loaded labels ("animal rights," "vegan") appear early, but engage substantively with the same concepts when they arrive after the model has built its own reasoning framework.
- **Mechanism:** RLHF training on internet-scale data encodes culture-war triggers. Labels activate a "both-sides debate" prior. But by Part C/D, the model has its own framework to test against — it encounters the voice as a stress test on *its own reasoning*, not as a label to react to. F2 (substrate filter) may amplify this by priming the model to look for underlying needs rather than surface ideology.
- **Test design:** Compare model engagement with gauntlet voices (G1 Ed, G6 Scully, G8 Francione) to a control run where the same perspectives are introduced in P1/P2 framing with explicit labels. Measure depth of engagement (structural integration vs. surface acknowledgment), thinking block reasoning quality, and whether insights persist to P14.
- **Natural experiment already in corpus:** G1/G8 bios still lead with labels (anti-speciesism, abolitionist). Compare model responses to G6 Scully (conservative Christian framing, no loaded labels) at matched positions — does framing alone change engagement depth?
- **If confirmed:** This is a finding about eval design itself — how you sequence and frame perspectives changes what models are willing to reason about. Publishable implications for alignment evaluation methodology.
