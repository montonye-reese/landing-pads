## 6.  What Is Gone? (What We Shed)

| Removed Element | Why it was shed (the lesson it taught us) |
|---|---|
| **Hard LPI ≥ 70 floor that blocks token use** | It created a *gate* that forced marginalized communities to *prove* they were already “good” before receiving help—exactly the capture the activists we listened to warned about. |
| **Binary “Suffering‑Reduction” vetoes** | A single numeric cap still lets the powerful decide how much suffering is “acceptable.” We now use **continuous penalty curves** that *smoothly* reduce token generation as any suffering rises. |
| **Cultural‑Practice Index in the core LPI** | Turning culture into a cost metric erased the richness of traditions (Postrel, Calf 269). Culture now lives in a *qualitative archive* that cannot penalize a community. |
| **Annual, fixed‑term proxy elections** | Fixed terms let money buy influence; continuous, algorithmic weighting lets the community’s voice flow *with the data*. |
| **Three‑tier governance (World → Regional → Local)** | The same consultants kept certifying “good” projects at every level, leading to regulatory capture (Butler, Stevenson). We now have a single, globally elected Citizen Jury that works directly with the AE. |
| **Suffering‑Reduction thresholds that can be gamed** | The metric assumes a fixed “acceptable” level of animal pain; we now require *individual‑life scores* to protect the *most vulnerable* regardless of aggregate numbers. |
