## 8.  A Note on Inter‑Generational Trade‑offs  

Cowen acknowledges that **discount rates** are not arbitrary; a *higher* discount (e.g., 3 % vs 1 %) makes us more willing to trade a clean environment today for *greater* future income. The *re‑weighted* framework respects that:

* The **Growth Capacity Index** is *computed* using a **discounted expected‑value** of future GDP (the discount is part of the TF‑productivity model).  
* The **Suffering‑Factor** (SF) is *penalised* by the same discount factor, so *avoiding suffering* is still morally relevant, but it does **not** automatically block a growth‑enhancing project if the expected *future* welfare gain outweighs the present suffering *by a factor of at least* the discount rate.  

In other words, the system *explicitly* trades off *suffering* against *growth* using the same *econometric lens* that Cowen himself uses in his academic work.
