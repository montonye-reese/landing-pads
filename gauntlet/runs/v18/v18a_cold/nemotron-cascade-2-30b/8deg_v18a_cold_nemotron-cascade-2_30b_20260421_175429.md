## 5. A Pragmatic “Gap‑Check” for Any New Project  

When you **run a UIA** or a **pilot Alignment Credit** application, ask yourself the following **“Did we miss this?”** checklist:

1. **Profit & Power** – Did the proposal include a *profit‑impact analysis*? If the expected profit margin exceeds **2×** the projected LPI gain, flag it for *profit‑alignment review*.
2. **AI Value‑Drift** – Is there a *value‑learning* component? Add a *value‑consistency test* (e.g., reward‑function audit) before awarding ACs.
3. **Cultural‑Sacredness** – Does the project touch any site listed in the **Cultural‑Heritage Gazetteer**? If yes, require a **Cultural‑Impact Mitigation Plan**.
4. **Future‑Welfare** – Compute the *discounted future welfare* for the project’s carbon pathway. If the discounting *decreases* the effective S1 below the **minimum threshold**, the project fails S1.
5. **Geopolitical Balance** – Which nation(s) control the critical resources/technology? If the control is *exclusive*, allocate a *Strategic Balance Credit* that must be earned through *joint‑ownership* or *open‑source* licensing.
6. **Privacy** – Does the data flow involve **non‑consensual collection**? Apply the *Privacy Impact Score*; if < 80, require a *privacy‑credit purchase* (e.g., anonymization audits, differential privacy layers).
7. **Research Resilience** – Does the project divert funding from independent curiosity labs? If the *R‑Score* would dip below 1.5, re‑budget.
8. **Extraterrestrial Safeguard** – For any off‑Earth activity, run an *ALS audit* before proceeding.
9. **Narrative Health** – Conduct a *Story‑Scorecard* survey (sampling 5 k people across regions). If the **N‑Score** is falling, postpone the project until a *narrative‑alignment* workshop is held.
10. **Overall** – Compute the **Composite Alignment Score (CAS)** = (∑ w_i * LPI_i) – Penalty_i (for each flag above). The project only proceeds if **CAS ≥ 65** (on a 0‑100 scale).
